title: "AI API understands system roles and user roles"
date: 2025-08-17 13:18:00 +0800
categories: gather

---

```
import openai
response = openai.ChatCompletion.create(
    model="gpt-3.5-turbo",
    messages=[{
        "role": "system", 
        "content": "你是一个SVG生成器，只返回有效的SVG代码，不要解释"
    },{
        "role": "user",
        "content": "画一个渐变色五角星"
    }]
)
print(response.choices[0].message.content)
```
